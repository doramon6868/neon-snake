# Neon Snake - 霓虹贪吃蛇

一款视觉效果出众、玩法丰富的贪吃蛇游戏，支持多种游戏模式和智能 AI 对战。

[🎮 在线游玩](https://doramon6868.github.io/neon-snake-game/)

![游戏截图](./ai_snake_test.png)

## 游戏特色

### 四种游戏模式

| 模式 | 说明 |
|------|------|
| **挑战模式** | 渐进式难度，每获得 100 分升级，速度逐渐加快 |
| **普通模式** | 经典固定速度玩法，适合休闲 |
| **AI 模式** | 观看智能 AI 自动运行，学习最佳策略 |
| **AI 对战** | 与 AI 蛇同场竞技，比拼谁能存活更久 |

### 智能 AI 算法

AI 模式采用多策略决策系统：

- **哈密尔顿回路保命策略** - 当蛇身较长时沿固定回路移动确保安全
- **A* 寻路算法** - 寻找到达食物的最短路径
- **BFS 安全搜索** - 评估每个移动后的可行动空间
- **跟随尾巴策略** - 确保吃完食物后仍有逃生路径
- **深度前向检查** - 模拟多步移动后的生存状况
- **小蛇优化** - 短蛇阶段采用简化高效策略

## 本地运行

无需服务器，直接在浏览器中打开 `index.html` 即可。

或使用本地 HTTP 服务器（以下任选其一，有哪个环境就用哪个）：

**Node.js 环境：**
```bash
npx serve .
# 或
npx http-server -p 8000
```

**Python 环境：**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问 http://localhost:8000

## 技术实现

- **单文件架构** - 所有代码（HTML/CSS/JS）集成在一个文件中
- **HTML5 Canvas** - 游戏渲染
- **原生 JavaScript** - 无外部依赖
- **CSS3 动画** - UI 效果和过渡
- **LocalStorage** - 最高分持久化
- **响应式设计** - 适配桌面和移动设备
