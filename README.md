# Neon Snake - 霓虹贪吃蛇

一个功能丰富的贪吃蛇游戏，使用 HTML5 Canvas 和 JavaScript 构建。

## 特性

### 游戏模式
- **挑战模式 (Challenge Mode)** - 渐进式难度，随着等级提升速度加快
- **普通模式 (Regular Mode)** - 经典固定速度玩法
- **AI 模式 (AI Mode)** - 观看 AI 自动玩贪吃蛇
- **AI 对战模式 (AI vs YOU)** - 与 AI 蛇在同一张地图上对战

### 水果与道具
- 7 种不同价值的水果：樱桃、草莓、橙子、猕猴桃、葡萄、西瓜、桃子
- 6 种特殊道具：
  - ⏰ 慢动作 - 游戏速度降低 50%
  - 🧺 水果篮 - 生成 10 个额外水果
  - 📏 直线水果 - 生成 5 个排成直线的水果
  - ❄️ 暴风雪 - 冰雪主题特效
  - 🥥 椰子树 - 热带主题特效
  - 🌸 樱花树 - 粉色主题特效

### 视觉效果
- 霓虹风格的现代化 UI 设计
- 丰富的粒子动画效果
- 蛇身吃水果的视觉反馈
- Power-up 全屏特效
- 菜单动画背景

### 移动端支持
- 响应式布局适配各种屏幕尺寸
- 触摸滑动控制
- 屏幕方向按钮（十字键布局）
- 触摸反馈效果
- 防止页面滚动干扰游戏

## 如何游玩

### 在线游玩
🎮 [点击这里在线游玩](https://doramon6868.github.io/neon-snake-game/)

### 本地运行
直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 控制方式
- **键盘**: 方向键 或 WASD
- **触摸**: 滑动屏幕
- **方向按钮**: 移动端屏幕上的十字方向键
- **菜单按钮**: 返回主菜单

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3 动画
- 无需外部依赖
